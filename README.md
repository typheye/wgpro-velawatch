# wgpro-velawatch

## 项目介绍
该版本腕管Pro适用于搭载Vela系统的小米智能手表设备运行。

## 适配情况

| 功能列表 | Redmi Watch 5 | Xiaomi Watch S3系列 | Xiaomi Watch S4系列 | Xiaomi Band 9 Pro |
| :----  | ----:  | ----: | ----: | ----: |
| 基础功能 | 支持 | 支持 | 支持 | 支持 |
| 腕上微聊 | 支持 | 支持 | 支持 | 支持 |
| X5浏览器 | * 支持 | * 支持 | * 支持 | |
| 位置信息 | 支持 | 支持 | 支持 | 支持 |
| 上传下载 | 支持 | 支持 | 支持 | |

注：带 * 功能需激活应用后支持。
## 项目编译

### 1. 开发

```
npm install
npm run start
```

### 2. 构建

```
npm run build
npm run release
```

### 3. 调试

```
npm run watch
```
