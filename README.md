# wgpro-velawatch

## 项目介绍
该版本腕管Pro适用于搭载Vela系统的小米智能手表设备运行。

## 适配情况

| 品牌 | 型号 | 设备类型  | 联网功能 | 定位功能 |
| ----:  | :----  | :----: | :----: | :----: |
| Redmi | Watch | 手表 | 支持 | 支持 |
| Xiaomi | Watch S3/S3 eSIM/Sport | 手表 | 支持 | 支持 |
| Xiaomi | Watch S4/S4 eSIM | 手表 | 支持 | 支持 |
| Xiaomi | Band 9 Pro | 手环 | - | - |

## 项目编译

### 1. 开发

```
npm install
npm run start
```

### 2. 构建

```
npm run build
npm run release
```

### 3. 调试

```
npm run watch
```
